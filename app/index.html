This is a sample express API used for testing json uploads

  • In order to view the files, go to files
  • In order to view the files, send a POST to /file-upload

POST Example:

  • endpoint: localhost:3000/file-upload
  • method: POST
  • Headers: { "Content-Type": "application/json", "ApiKey": "1234" }
  • Body: { "name": "john", "surname": doe }
